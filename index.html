Getting Started

Required Components

  • Node.js
  • LiveReload for Chrome / for Firefox

Set Up

$ sudo npm install -g grunt-cli
$ git clone git@github.com:sanographix/rin-html-template.git test
$ cd test
$ npm install
$ grunt

Local Server

Visit http://localhost:8000/.
